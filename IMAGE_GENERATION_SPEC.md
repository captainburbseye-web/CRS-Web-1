# CRS Rack Module Images - Generation Spec

## Overview
Generate 4 photorealistic rack-mounted hardware module images for CRS website.

---

## GLOBAL STYLE REQUIREMENTS

**Format:**
- 19" rack mount standard width
- Aspect ratio: **16:9** or **21:9** (wide horizontal)
- Resolution: **2000px width minimum**
- Format: PNG with **solid black background** (#000000)

**Aesthetic:**
- Photorealistic hardware (1970s-1990s studio gear)
- Dark metal panels (brushed aluminum, black anodized)
- Rack ears visible on LEFT and RIGHT sides
- Mounting screws in corners (Phillips head)
- Professional broadcast/studio quality
- NO plastic, NO consumer gear look

**Lighting:**
- Top-down studio lighting
- Slight shadow at bottom edge of panel
- Highlights on metal edges and knobs
- LEDs should glow realistically (not flat)
